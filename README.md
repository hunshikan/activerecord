
# JFINAL ACTIVERECORD
====

Peel from jfinal

## How to use

**maven**

add repository to pom

```xml
<repositories>
  <repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
  </repository>
</repositories>
```

add dependency

```xml
<dependency>
  <groupId>com.github.iaceob</groupId>
  <artifactId>activerecord</artifactId>
  <version>3.3</version>
</dependency>
```

**more**

see [activerecord jitpack](https://jitpack.io/#iaceob/activerecord/)
